Adaptive automatically switches to backup providers when your primary provider fails. No downtime, no interruptions.

## Fallback Configuration

Fallback is **disabled by default** for cost control. Enable it by specifying a `mode` when you need maximum reliability and can handle higher costs.

### Default Behavior (Disabled)
```javascript
const completion = await openai.chat.completions.create({
  model: "",
  messages: [{ role: "user", content: "Hello" }]
  // No fallback configuration or empty mode = disabled
});
```

### Enable Fallback
```javascript
const completion = await openai.chat.completions.create({
  model: "",
  messages: [{ role: "user", content: "Hello" }],
  fallback: {
    mode: "race"  // Enable by specifying mode
  }
});
```

### Explicitly Disable Fallback
```javascript
const completion = await openai.chat.completions.create({
  model: "",
  messages: [{ role: "user", content: "Hello" }],
  fallback: {
    mode: ""  // Explicitly disabled with empty mode (same as omitting)
  }
});
```

## Fallback Modes

### Race Mode
Try multiple providers at once for maximum speed and reliability:

```javascript
const completion = await openai.chat.completions.create({
  model: "",
  messages: [{ role: "user", content: "Hello" }],
  fallback: {
    mode: "race"
  }
});
```

**Pros:** Ultra-low latency, max reliability  
**Cons:** Higher cost (multiple calls)

### Sequential Mode
Try providers one by one:

```javascript
const completion = await openai.chat.completions.create({
  model: "",
  messages: [{ role: "user", content: "Hello" }],
  fallback: {
    mode: "sequential"
  }
});
```

**Pros:** Lower cost, predictable  
**Cons:** Higher latency on failures

## Performance

- **99.95%** uptime across providers
- **Under 500ms** failover latency  
- **Under 30s** recovery time

## Error Handling

```javascript
try {
  const completion = await openai.chat.completions.create({
    model: "",
    messages: [{ role: "user", content: "Hello" }],
    fallback: {
      mode: "sequential"
    }
  });
} catch (error) {
  if (error.code === 'all_providers_failed') {
    console.log('All providers unavailable');
  }
}
```
