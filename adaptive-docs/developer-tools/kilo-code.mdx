**Save 60-80% on AI development costs** with intelligent model routing. Works seamlessly with Kilo Code's development features.
</Tip>

## Get Your Adaptive API Key

Visit [llmadaptive.uk](https://www.llmadaptive.uk) to create an account and generate your API key.

## Quick Setup

<Steps>
  <Step title="Install Kilo Code" icon="download">
    Follow Kilo Code's installation instructions for your platform and ensure it's properly configured.
  </Step>
  <Step title="Configure API Settings" icon="gear">
    Set up Adaptive API configuration:
    ```bash
    # Set environment variables
    export OPENAI_BASE_URL=https://www.llmadaptive.uk/api/v1
    export OPENAI_API_KEY=your-adaptive-api-key
    ```
  </Step>
  <Step title="Start Coding" icon="play">
    Launch Kilo Code and begin using AI-assisted development with cost-optimized routing.
  </Step>
</Steps>

## Alternative Setup Methods

<CodeGroup>
```bash Environment Variables
export OPENAI_BASE_URL=https://www.llmadaptive.uk/api/v1
export OPENAI_API_KEY=your-adaptive-api-key
kilo-code
```

```json Configuration File
// In kilo-code config
{
  "api": {
    "provider": "openai-compatible",
    "baseUrl": "https://www.llmadaptive.uk/api/v1",
    "apiKey": "your-adaptive-api-key"
  }
}
```

```text Manual Configuration
1. Open Kilo Code settings
2. Set API Provider to "OpenAI Compatible"
3. Enter Base URL and API key as shown above
```
</CodeGroup>

## Troubleshooting

<AccordionGroup>
<Accordion title="Connection Issues" icon="wifi-off">
- Verify your API key at [llmadaptive.uk/dashboard](https://www.llmadaptive.uk/dashboard)
- Check that Base URL is set to `https://www.llmadaptive.uk/api/v1`
- Restart Kilo Code after configuration changes
</Accordion>

<Accordion title="Configuration Errors" icon="gear">
- Ensure environment variables are properly set
- Verify configuration file syntax if using file-based config
- Test connection with a simple request after setup
</Accordion>
</AccordionGroup>
