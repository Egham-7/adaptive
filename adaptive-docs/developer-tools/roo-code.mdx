</Tip>

## Get Your Adaptive API Key

Visit [llmadaptive.uk](https://www.llmadaptive.uk) to create an account and generate your API key.

## Quick Setup

<Steps>
  <Step title="Install Roo Code Extension" icon="download">
    1. Open VS Code Extensions Marketplace
    2. Search for "Roo Code" and install the official extension
    3. Click the Roo Code icon in the sidebar
  </Step>
  <Step title="Configure API Settings" icon="gear">
    Navigate to Roo Code settings and enter:
    ```text
    API Provider: OpenAI Compatible
    Base URL: https://www.llmadaptive.uk/api/v1
    API Key: your-adaptive-api-key
    Model: [leave empty for intelligent routing]
    ```
  </Step>
  <Step title="Start Project Analysis" icon="play">
    Use Roo Code's project analysis, code generation, and refactoring features with cost-optimized routing.
  </Step>
</Steps>

## Alternative Setup Methods

<CodeGroup>
```json Settings File
// In VS Code settings.json
{
  "rooCode.provider": "openai-compatible",
  "rooCode.baseUrl": "https://www.llmadaptive.uk/api/v1",
  "rooCode.apiKey": "your-adaptive-api-key"
}
```

```bash Environment Variables
export OPENAI_BASE_URL=https://www.llmadaptive.uk/api/v1
export OPENAI_API_KEY=your-adaptive-api-key
```

```text Manual Configuration
1. Open Roo Code settings in VS Code
2. Set API Provider to "OpenAI Compatible"
3. Enter Base URL and API key as shown above
```
</CodeGroup>

## Troubleshooting

<AccordionGroup>
<Accordion title="Connection Issues" icon="wifi-off">
- Verify your API key at [llmadaptive.uk/dashboard](https://www.llmadaptive.uk/dashboard)
- Check that Base URL is set to `https://www.llmadaptive.uk/api/v1`
- Restart VS Code after configuration changes
</Accordion>

<Accordion title="Project Analysis" icon="magnifying-glass">
- Ensure project is properly opened in VS Code workspace
- Leave model field empty for intelligent routing (recommended)
- Test with a simple analysis task after setup
</Accordion>
</AccordionGroup>
