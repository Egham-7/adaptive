## 1. Get Your API Key

1. Sign up at [llmadaptive.uk](https://llmadaptive.uk)
2. Create a project and generate an API key

## 2. Update Your Code

Just change the base URL - that's it!

<CodeGroup>

```javascript JavaScript/TypeScript
import OpenAI from 'openai';

const openai = new OpenAI({
  apiKey: 'your-adaptive-api-key',
  baseURL: 'https://llmadaptive.uk/api/v1'  // Only change needed
});

const completion = await openai.chat.completions.create({
  model: '', // Empty = intelligent routing
  messages: [{ role: 'user', content: 'Hello!' }]
});
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="your-adaptive-api-key",
    base_url="https://llmadaptive.uk/api/v1"  # Only change needed
)

completion = client.chat.completions.create(
    model="",  # Empty = intelligent routing
    messages=[{"role": "user", "content": "Hello!"}]
)
```

```bash cURL
curl https://llmadaptive.uk/api/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "X-Stainless-API-Key: your-adaptive-api-key" \
  -d '{"model": "", "messages": [{"role": "user", "content": "Hello!"}]}'
```

</CodeGroup>

## 3. Test It Out

Run your code and see Adaptive in action. The response includes a `provider` field showing which AI provider was automatically selected:

```json
{
  "id": "chatcmpl-123",
  "choices": [{"message": {"content": "Hello! How can I help you?"}}],
  "provider": "gemini",  // ‚Üê Adaptive automatically chose Gemini
  "usage": {"total_tokens": 15}
}
```

## Key Differences from OpenAI

| Aspect | OpenAI | Adaptive |
|--------|--------|----------|
| **Base URL** | `api.openai.com/v1` | `llmadaptive.uk/api/v1` |
| **Model** | `"gpt-4"` (specific) | `""` (intelligent routing) |
| **Providers** | OpenAI only | 6+ providers |
| **Cost** | Fixed pricing | Up to 80% savings |
