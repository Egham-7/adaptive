## Get Your API Key

1. Sign up at [llmadaptive.uk](https://llmadaptive.uk)
2. Create a project and generate an API key

<Warning>
Keep your API key secure. Never expose it in client-side code.
</Warning>

## Use Your API Key

Works with any OpenAI-compatible header:

<CodeGroup>

```bash cURL
curl https://llmadaptive.uk/api/v1/chat/completions \
  -H "X-Stainless-API-Key: your-adaptive-api-key" \
  -H "Content-Type: application/json" \
  -d '{"model": "", "messages": [{"role": "user", "content": "Hello!"}]}'
```

```javascript JavaScript
import OpenAI from "openai";

const openai = new OpenAI({
  apiKey: process.env.ADAPTIVE_API_KEY,
  baseURL: "https://llmadaptive.uk/api/v1"
});
```

```python Python
import os
from openai import OpenAI

client = OpenAI(
    api_key=os.getenv("ADAPTIVE_API_KEY"),
    base_url="https://llmadaptive.uk/api/v1"
)
```

</CodeGroup>

## Rate Limits

- **Free**: 100 requests/minute
- **Pro**: 1,000 requests/minute  
- **Enterprise**: Custom limits

## Error Codes

- **401**: Invalid API key
- **429**: Rate limit exceeded
